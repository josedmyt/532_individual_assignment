# Spotifind in Shiny for R

This app is a re-implementation of our group Spotify dashboard, rebuilt in Shiny for R.

## Features
- Filter songs by genre, popularity, and danceability
- View summary metrics
- Explore top genres in a bar chart
- Browse filtered songs in a table
